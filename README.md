# Analog Clock
## Overview
A simple desktop analog clock application built with Python, PySimpleGUI, and Tkinter Canvas.

## Screenshot
<p align="center">
  <img src="screenshot.png" width="300">
</p>

## Features
- Displays the current time with an analog clock
- Updates every 100 ms
- Draws hour, minute, and second hands
- Draws hour markers using trigonometric functions

## Technologies
- Python
- PySimpleGUI
- Tkinter Canvas

## How to Run
1. Install the required package.

```bash
pip install PySimpleGUI
```

2. Run the application.

```bash
python analog_clock.py
```

## What I learned
This project helped me understand:
- How to update a GUI repeatedly
- How to draw shapes on a Canvas
- How to calculate positions with trigonometric functions
- How to split a program into reusable functions
