# Analog Clock
## Overview
A simple desktop analog clock application built with Python, PySimpleGUI, and Tkinter Canvas.
Through this project, I learned the basics of GUI programming, how to draw on a Canvas, and how to update the clock based on the current time.

## Screenshot
<p align="center">

</p>

## Features
- Displays the current time as an analog clock
- Updates the display every 100 ms
- Draws the hour, minute, and second hands in real time
- Draws the hour markers using trigonometric functions

## Technologies
- Python
- PySimpleGUI
- Tkinter Canvas

## How to Run
1. Install the required package.

```bash
pip install PySimpleGUI
```

2. Run the application.

```bash
python analog_clock.py
```

## What I learned
- How to create an analog clock using PySimpleGUI and Canvas
- How to draw shapes by calculating X and Y coordinates
- How to calculate the positions of the clock hands and hour markers using trigonometric functions
- How to calculate the angles of the hour, minute, and second hands from the current time

## Future Improvements
- Play an alarm every hour
- Add a background image
- Display a digital clock

## 🇯🇵 日本語

### 概要
Python・PySimpleGUI・Tkinter Canvasを使用して、アナログ時計を表示するデスクトップアプリを作成しました。
このアプリを通して、GUIの基本的な作り方や、Canvasへの描画、現在時刻に合わせて時計を更新する方法を学びました。

### 機能
- 現在時刻をアナログ時計で表示
- 100msごとに画面を更新
- 時針・分針・秒針をリアルタイムで描画
- 三角関数を用いて文字盤の目盛りを描画

### 使用技術
- Python
- PySimpleGUI
- Tkinter Canvas

### 学習したこと
- PySimpleGUIとCanvasを使ってアナログ時計を表示する方法
- X座標・Y座標を計算して図形を描画する方法
- 三角関数を使って時計の針や目盛りの位置を計算する方法
- 現在時刻から時針・分針・秒針の角度を求める方法
